# Hookah Tobacco Price Tracker

A Python-based web scraper that monitors prices and weight of tobacco products from hookahhub.
It calculates the price per gram and exports a sorted ranking to an Excel file to help find the best value deals.

## Features
- Scrapes multiple pages automatically.
- Extracts product names, prices, and weights (handling different units).
- Calculates **Price per Gram**.
- Exports data to `tobacco_ranking.xlsx` sorted by the best value.
